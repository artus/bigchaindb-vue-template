# bigchaindb-vue-frontend

> A vue.js template for creating blockchain applications with BigchainDB.

## Build Setup

Simply run the start script:

```shell
chmod +x ./start.sh ./stop.sh ./down.sh;
./start.sh;
```

To stop the service:

```shell
./stop.sh;
```

To remove the service:

```shell
./down.sh;
```
